# Neuro-Prompting Protocol (NPP)

Official repository for the **Neuro-Prompting Protocol (NPP)**, a framework designed to optimize human-machine interaction through the synergistic use of NLP-driven Attention Steering and Bayesian Context Pruning.

## Overview
The NPP protocol introduces a structural separation between logic (SOTA LLM) and memory management (local SLM). It leverages Neuro-Linguistic Programming (NLP) patterns not as psychological tools, but as heuristic frameworks to mathematically guide the attention matrices of Transformers towards high-quality latent space regions.

## Key Features
- **Semantic Framing:** Using NLP schemas for Attention Steering.
- **Bayesian Dynamic Context:** Statistical pruning of local memory to maximize Signal-to-Noise Ratio (SNR).
- **Linguistic Normalization:** Centralized translation to eliminate multi-language bias.
